# stm32cubeide_guide
도무지 STM32cubeide의 설정값에 대한 설명과 데이터시트상의 register가 매치가 안되서 작성함

 해당 문서는 STM32G474RE를 기준으로 제작함 따라서 다른 MCU모델에 동일하게 적용이 되지 않을 수 있음을 알림

 주로 사용하는 기능을 우선적으로 작성하기에 없는 항목이 있을 수 있음

 예제 코드를 작성하기 위한 목적이 아님
